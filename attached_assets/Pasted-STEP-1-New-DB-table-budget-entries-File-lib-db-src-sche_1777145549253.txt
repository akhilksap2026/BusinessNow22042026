STEP 1 — New DB table: budget_entries
File: lib/db/src/schema/financials.ts or new lib/db/src/schema/budgetEntries.ts
(export from lib/db/src/schema/index.ts)

export const budgetEntriesTable = pgTable("budget_entries", {
  id:            serial("id").primaryKey(),
  projectId:     integer("project_id").notNull(),
  entryDate:     text("entry_date").notNull(),
  type:          text("type").notNull(),          -- 'SOW' | 'CO' | 'Adjustment'
  description:   text("description").notNull(),
  amount:        numeric("amount",  { precision: 15, scale: 2 }).notNull().default("0"),
  hours:         numeric("hours",   { precision: 10, scale: 2 }).notNull().default("0"),
  documentLink:  text("document_link"),
  changeOrderId: integer("change_order_id"),
  createdAt:     timestamp("created_at", { withTimezone: true }).notNull().defaultNow(),
});

Run: cd lib/db && npm run push

STEP 2 — New API routes
File: artifacts/api-server/src/routes/projects.ts

Route A: GET /projects/:id/budget-entries
- Fetch all budget_entries WHERE project_id = :id, ORDER BY entry_date ASC.
- Compute running totals per row: runningAmount, runningHours.
- Return: { totalAmount, totalHours, entries: [...with running totals] }

Route B: POST /projects/:id/budget-entries  (requirePM)
- Body: { entryDate, type, description, amount, hours, documentLink? }
- Validate: type in ['SOW','CO','Adjustment'], description required.
- INSERT row. Return created entry. Log audit.

Auto-insert on CO approval:
- In the PATCH handler that sets a change order to "Approved":
  If changeOrder.amount > 0 or changeOrder.additionalHours > 0:
    INSERT budget_entry { type:'CO', amount: changeOrder.amount,
      hours: changeOrder.additionalHours, changeOrderId: changeOrder.id }
  Also UPDATE projects SET budget = budget + changeOrder.amount,
    budgeted_hours = budgeted_hours + changeOrder.additionalHours.
