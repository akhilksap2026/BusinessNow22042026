STEP 4 — Backend: POST /projects/:id/apply-template  (requirePM)
Body: { templateId: number, phaseName?: string }

1. Fetch project (404 if missing). Fetch template with phases+tasks.
2. Determine phase name: body.phaseName ?? template.name.
3. INSERT a phase task (isPhase=true) on the project. Capture phaseTaskId.
4. For each template task (top-level first, parentTaskId=null):
   Calculate start/due dates from project.startDate + relative offsets.
   INSERT into tasks (projectId, parentTaskId=phaseTaskId, name, status='Not Started',
   priority, effort, billable, startDate, dueDate, fromTemplate=true, appliedTemplateId).
   Store templateTaskId → real taskId map.
5. For child template tasks (parentTaskId != null):
   Set parentTaskId = taskIdMap[templateTask.parentTaskId].
6. Log audit. Return { phaseTaskId, phasesCreated: 1, tasksCreated: N }.
