Problem:

Please carefully inspect the app layout before making changes. Look especially for shared layout/page wrapper components, sidebar layout code, and Tailwind/CSS classes such as:

- max-w-screen-xl
- container
- mx-auto
- px-6, px-8, p-6, p-8
- py-6, py-8
- space-y-6
- gap-6
- margin-left values for the sidebar
- fixed content widths

Main goal:
Make the app use more of the available viewport width and reduce unnecessary vertical spacing, while keeping the UI clean and readable.

  - collapsed sidebar: about 48px to 64px
